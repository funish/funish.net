# Funish.net

![GitHub](https://img.shields.io/github/license/funish/funish.net)
[![Contributor Covenant](https://img.shields.io/badge/Contributor%20Covenant-2.1-4baaaa.svg)](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)

> Discover and search packages across NPM and Winget registries in one place.

## Features

- 🔍 **Unified Search**: Search NPM and Winget packages with sorting, filtering, and pagination
- 📦 **Package Details**: README, versions, dependencies, file tree browser for NPM packages
- 🪟 **Winget Support**: Package description, version history, and installer downloads
- 👥 **Org/User Pages**: Browse packages by npm organization or user with on-demand metadata
- 🌟 **Explore**: Trending NPM packages and featured Winget applications
- 🌐 **i18n**: English and Simplified Chinese

## Quick Start

### Prerequisites

- Node.js 22+
- pnpm 10+

### Installation

```bash
git clone https://github.com/funish/funish.net.git
cd funish.net
pnpm install
```

### Development

```bash
pnpm dev
```

The application will start on `http://localhost:3000`.

### Build

```bash
pnpm build
pnpm preview
```
